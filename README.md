# Jo's Kitchen – Hotel Menu Web Application

## Overview
Jo's Kitchen is a responsive restaurant menu web application developed as part of a Full Stack Development (FSD) lab project.

The project was gradually enhanced using HTML, CSS, and JavaScript to create an interactive and visually appealing food ordering experience inspired by South Indian cuisine themes.

## Features
- Responsive restaurant menu layout
- Categorized food sections
- Interactive quantity controls
- Dynamic grand total calculation
- Bill generation system
- Styled UI with custom color themes and typography
- Smooth and user-friendly interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Functionalities
- Add or remove item quantities
- Automatically update total bill amount
- Generate order summary with selected items
- Responsive grid-based design for different screen sizes

## Project Type
Academic Full Stack Development Lab Project

## Learning Outcomes
This project helped improve:
- Frontend web development skills
- JavaScript DOM manipulation
- Responsive web design
- UI styling and layout structuring
- Interactive webpage functionality

## Note
This project was created as part of academic lab exercises and gradually improved throughout the learning process.
